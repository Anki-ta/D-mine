DDL

CREATE DATABASE

Description: This statement is used to create a new SQL database.
Syntax:
CREATE DATABASE databasename;
Example:
CREATE DATABASE rait_db;

The above SQL statement creates a database called 'rait_db'.

CREATE TABLE

Description: The statement allows you to create and define a table.
Syntax:
CREATE TABLE table_name
(
column1 datatype [ NULL | NOT NULL ],
column2 datatype [ NULL | NOT NULL ],
...
);
Example:
CREATE TABLE students
(
stud_idint NOT NULL,
stud_namechar(50) NOT NULL,
stud_noint(10)
);

The example creates a table called students which has 3 columns.
  1. The first column is called stud_id which is created as a number datatype (maximum 10 digits in length) and cannot contain null values.
  2. The second column is called stud_name which is a char datatype (50 maximum characters in length) and also can not contain null values.
  3. The third column is called stud_no which is a int datatype but can contain null values.

ALTER TABLE

Description: The SQL ALTER TABLE statement is used to add, modify, or drop/delete columns in a table. The SQL ALTER TABLE statement is also used to rename a table.
Add column in table
Syntax:
ALTER TABLE table_name ADD column_name column-definition;

Example:
ALTER TABLE student ADD stud_namechar(30);

This SQL ALTER TABLE example will add a column called stud_name with char datatype(30 maximum characters in length) to the student table.
Modify column in table
Syntax:
ALTER TABLE tablename CHANGE old_column_name new_column_name column_type;

Example:
ALTER TABLE student CHANGE roll_no sr_no INT(4);

This SQL ALTER TABLE example will modify a column called roll_no in the student table with the name sr_no and INT datatype(maximum length 4).
Drop column in table
Syntax:
ALTER TABLE tablename DROP COLUMN column_name;

Example:
ALTER TABLE student DROP COLUMN phn_no;

This SQL ALTER TABLE example will drop a column called phn_no from the student table.
Rename a table
Syntax:
ALTER TABLE old_table RENAME TO new_table;

Example:
ALTER TABLE stud_table RENAME TO student;

This example would rename the stud_table to student.

DROP DATABASE

Description: This statement is used to drop an existing SQL database.
Syntax:
DROP DATABASE databasename;

Example:
DROP DATABASE rait_db;

The above SQL statement drops the existing database "rait_db".

DROP TABLE

Description: This statement is used to an existing table.
Syntax:
DROP TABLE table_name;

Example:
DROP TABLE students;

The above SQL statement drops the existing table 'students'.
Note: This would both remove the records associated with the 'students' table as well as its table definition.

TRUNCATE TABLE

Description:The SQL TRUNCATE TABLE statement is used to remove all records from a table.
Syntax:
TRUNCATE TABLE table_name;

Example:
TRUNCATE TABLE student;

This example would truncate the table called student and remove all records from that table.

DML

INSERT

Description: This statement is used to insert a one or more records into a table. There are 2 syntaxes for the INSERT statement depending on whether you are inserting one record or multiple records.
Syntax:
INSERT INTO table (column1, column2, ... ) VALUES (expression1, expression2, ... );

Or the syntax for the INSERT statement when inserting multiple records in SQL is:
INSERT INTO table (column1, column2, ... ) SELECT expression1, expression2, ... FROM source_tables [WHERE conditions];
Parameters or Arguments:
table
The table in which to insert the records.
column1, column2
These are the columns in the table to insert values.
expression1, expression
These are the values to assign to the columns in the table. So column1 would be assigned the value of expression1, column2 would be assigned the value of expression2, and so on.
source_tables
Used when inserting records from another table. This is the source table when performing the insert.
WHERE conditions
Optional. Used when inserting records from another table. These are the conditions that must be met for the records to be inserted.
Example: Using INSERT Statement to Insert One Record The simplest way use the INSERT statement is to insert one record into a table using the VALUES keyword. Let's look at an example of how to do this in SQL. In this example, we have a table called categories with the following data:

category_id category_name
25 Deli
50 Produce
75 Bakery
100 General Merchandise
125 Technology

To insert a new category record. Enter the following SQL statement:
INSERT INTO categories (category_id, category_name) VALUES (150, 'Miscellaneous');

There will be 1 record inserted. Select the data from the categories table again:
SELECT * FROM categories;

These are the results that you should see:
category_id category_name
25 Deli
50 Produce
75 Bakery
100 General Merchandise
125 Technology
150 Miscellaneous

UPDATE

Description: The SQL UPDATE statement is used to update existing records in the tables.
Syntax:
UPDATE table SET
column1 = expression1,
column2 = expression2,
...
[WHERE conditions];

Here the WHERE conditions are optional and they mean the conditions that must be met for the update to execute. If no conditions are provided, then all records in the table will be updated.
Example:
UPDATE student SET sr_no = 20 WHERE stud_name='Namita Khanna' ;

This example would update the record from the student table and set the sr_no as '20' where the student name is 'Namita Khanna'.

DELETE

Description: The SQL DELETE statement is a used to delete one or more records from a table.
Syntax:
DELETE FROM table_name [WHERE conditions];

Here the WHERE conditions are optional and they mean the conditions that must be met for the records to be deleted. If no conditions are provided, all records in the table will be deleted.
Example:
DELETE FROM student WHERE sr_no = 20;

This example would delete the record from the student table where the sr_no is 20.

SELECT

Description: The SQL SELECT statement is used to display content of a particular table. Content is shown using some conditions.
Syntax:
SELECT column_names FROM table_name WHERE conditions;
Here the WHERE conditions are optional and they mean the conditions that must be met for the records to be shown. If no conditions are provided, all records in the table will be shown.
Example: Using SELECT Statement to select data The simplest way use the SELECT statement is to select entire table using *. Let's look at an example of how to do this in SQL. In this example, we have a table called categories with the following data:

category_id category_name
25 Deli
50 Produce
75 Bakery
100 General Merchandise
125 Technology

Now to display entire content, the simplest SELECT query will be..
SELECT * FROM categories;

Thus it will show entire table. Now selecting particular row from table using WHERE clause..
SELECT * FROM categories WHERE category_id = 75;
The results are as follows..
category_id category_name
75 Bakery

As shown, only one row gets selected!
